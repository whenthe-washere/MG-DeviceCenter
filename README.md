# Microgeometrics. Device Center
See info about your Microvision or GeoVision device, do tests(e.g.: dead pixels, artifacting, benchmarking etc.) on them, recieve live support for your device and chat with other Microvision or GeoVision users directly from this app.
This app is still being developed as of this moment, and is not ready to be released(Release date is expected to be the same day Microvision and GeoVision devices finish being manufactured and start being sold.).
